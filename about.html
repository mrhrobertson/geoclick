What is this all about?

I will be developing a website with a interactive world map and a dynamic form to create an online quiz, that will serve questions that either require interaction with the world map through a text or image-based question, a multiple choice selection, or a text-input answer with some lexical leniency.

This is not a unique concept, as outlined in my research, however many of the current implementations rely on emulations, such as Java Runtime Environment (JRE) and Flash Shockwave Player, or non-standard libraries to create this functionality. My aim is to create a platform-agnostic version of these quizzes that can handle both desktop and mobile use whilst using HTML5, CSS3 and vanilla JavaScript.
